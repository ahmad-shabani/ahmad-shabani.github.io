# Ahmad Shabani Academic Website

Static GitHub Pages website for `ahmad-shabani.github.io`.

## Publish on GitHub Pages

1. Create a GitHub repository named exactly: `ahmad-shabani.github.io`
2. Upload all files and folders from this package into the root of that repository.
3. Commit the files to the `main` branch.
4. Go to repository **Settings → Pages**.
5. Set **Source** to **Deploy from a branch**.
6. Choose **Branch: main** and **Folder: /root**.
7. Save and open: `https://ahmad-shabani.github.io/`
